Backgrid.js

Backgrid.js is a set of components for building semantic and easily stylable tabular data grid widgets while offering a simple, intuitive and beautiful programming interface that doesn't put any handcuffs on you.

See Examples Download (v0.1)

Installation

Since Backgrid.js is built on Backbone.js, there are a number of dependencies you have to satisfy before you can start using it.

At the mininum, you have to get the following libraries first and put them into the <head> section of your HTML page.

  • jquery >= 1.7.0
  • underscore.js > 1.4.0
  • backbone.js >= 0.9.2
  • underscore.string.js >= 2.3.0

If you don't care about getting the latest version of the dependencies, you can use the bundled version of the libraries in the downloaded package.
