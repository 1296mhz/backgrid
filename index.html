Backgrid.js

Backgrid.js is a set of components for building semantic and easily stylable tabular data grid widgets.

It offers you a simple, intuitive and beautiful programming interface that doesn't put any handcuffs on you.

See Examples Download (v0.1)

Features

Advantages

  • No Hungarian notations.
  • Solid foundation. Based on Backbone.js.
  • Easily stylable. Just style with plain CSS like you would a normal HTML table.
  • Low learning curve. Works with plain old Backbone models and collections, no extra patterns or abstractions to learn.
  • Highly modular and customizable. Componenets are just simple Backbone View classes, customization is easy if you already know Backbone.
  • Lightweight. Extra features are separated into extensions, which keeps the bloat away.
  • Good documentation.

Supported browsers: [1]

  • Internet Explorer 8+
  • Chrome 4+
  • Safari 4+
  • Firefox 4+
  • Opera 9+ [3]

Notes:

1. Both the desktop and mobile versions of the above browsers are supported.
2. Currently needs to double click on a cell to gain focus.
